# B2B 获客与企业信息平台（网站版）

这是一个面向销售人员、企业老板、渠道商的 B2B 网站平台，
用于整合企业信息、行业客户资源，并提供搜索、筛选和智能推荐能力。

## 项目目标
- 搭建一个可搜索的企业信息库
- 帮助用户按行业 / 地区寻找潜在客户
- 支持企业认领、名片展示和商机发布
- 构建合规、可持续的获客平台（网站版）

## 核心功能（MVP）
- 首页（行业入口 + 搜索）
- 企业库（列表 / 详情页）
- 企业认领
- 智能推荐（同业 / 同地区）
- 用户名片页
- 商机发布
- 会员权限控制

## 技术方向（规划）
- 前端：Next.js 或 Vue
- 后端：Node.js（NestJS）或 Django
- 数据库：MySQL / PostgreSQL

## 合规说明
- 企业信息来自用户提交、企业认领及公开资料
- 联系方式默认脱敏展示
- 不涉及非法数据抓取或隐私数据买卖

## 项目状态
- 当前阶段：产品设计 / MVP 规划
- 后续：选择开源项目作为基础进行开发
